<h1 class="page-title">Acceptance Tests — Anthropic Messages</h1>

Test any API endpoint for Anthropic Messages compliance.
Configure your connection details and run the test suite to validate
responses against the Anthropic Messages schema. The suite defaults to
the `x-api-key` auth header (no Bearer prefix) and injects the
`anthropic-version: 2023-06-01` header automatically.

Provide your Anthropic API key in the form below — `OPENAI_API_KEY` is
not used for this suite.

<ComplianceTester
  client:load
  suite={anthropicMessagesSuite}
  defaultApiKey={import.meta.env.ANTHROPIC_API_KEY ?? ""}
/>
